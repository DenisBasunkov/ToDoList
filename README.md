# Fullstack Developer Техническое задание

Задание выполнено с 1 по 4 уровень. 5 уровень не выполнен так как никогда с ним не работал и не успел бы изучить.

Программу можно запустить через [*ссылку на деплой*](https://denisbasunkov.github.io/ToDoList/).

Либо командой `npm run dev` локально. 

Сервер также [*задеплоен на сайте RENDER.com*](https://todolistserver-9yzt.onrender.com). 
* [*Ссылка на репозиторий сервера*](https://github.com/DenisBasunkov/ToDoListServer)

Или командой `npm start`. 

После входа есть возможность войти уже зарегистированным пользователем ***(Демо данные: Логин: Test, Пароль: 12345)*** или создать нового пользователя.

В программе реализованны почти все доработки из уровня 2.
* Изменение очередности задач с помощью Drag&Drop, есть небольшой недочет во время редактирования и просмотра карточки задачи, в этих формаж нельзы выделить текст. Это связано с бибилотекой React-dnd, я с ней впервые работаю поэтому вышло немного так, но карточки спокойно можно переносить как внутири одного списка, так и в другие списки.
